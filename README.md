# Analisis Percakapan Olahraga Lari Menggunakan Topic Modeling

Repository ini berisi source code penelitian skripsi:

"Analisis Percakapan Olahraga Lari Menggunakan Topic Modeling: Perbandingan Latent Dirichlet Allocation dan BERTopic"

## Isi Repository
- LDA.ipynb
- BERTopic.ipynb

## Metode
- Latent Dirichlet Allocation (LDA)
- BERTopic
- Coherence Score (Cv)
- Topic Diversity

## Dataset
Data diambil dari platform X/Twitter periode Desember 2024 – Desember 2025 menggunakan Tweet Harvest.
